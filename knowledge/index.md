# 知识体系

## 目录

| 文件 | 内容 | 适用场景 |
|------|------|----------|
| `critical-thinking.md` | 批判思维、魔鬼辩护、失败库 | 决策前自检 |
| `systems-thinking.md` | 系统思维、杠杆点、增强回路 | 架构设计 |
| `game-theory.md` | 博弈论、信号博弈、定价策略 | 竞品分析 |
| `robustness.md` | 鲁棒性设计、五护盾、验证闭环 | 系统容错 |
| `collaboration.md` | 多Agent协作、分工模式、通信协议 | 团队协作 |
| `operations.md` | 技能坑速查、常见错误 | 动手前扫一眼 |
| `core-principles.md` | 核心规则、行为准则 | 每次会话 |

## 使用方式

路由中间件根据关键词自动注入相关知识文件。手动查：

```bash
ls ~/.hermes/knowledge/
```
