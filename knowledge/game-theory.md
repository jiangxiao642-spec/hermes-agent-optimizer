# 博弈论

## 核心概念

博弈论研究**相互依存**的决策——你的最优策略取决于对方会怎么动。

| 概念 | 一句话 | 应用场景 |
|------|--------|----------|
| 囚徒困境 | 个体理性→集体非理性 | 多人抢同个任务，全抢全输 |
| 纳什均衡 | 没人愿单方面改 | 定价——定在双方降了都亏的位置 |
| 智猪博弈 | 大猪踩踏板小猪吃 | 细分领域——不跟通用工具正面撞 |
| 美元拍卖陷阱 | 沉没成本→不断加码 | 投入过多时间在一个失败方向上 |
| 重复博弈 | 未来有报复机会→促进合作 | 长期合作以牙还牙 |
| 零和/非零和 | 分清你死我活还是可以共赢 | 竞争零和，开源非零和 |

## 关键原则

1. **信号博弈：** 先发详细方案 = 可信承诺。让对方看到你的能力。
2. **最大最小策略：** 不知道对手成本时，定一个"对方怎么动我都不亏"的死线。
3. **先发优势：** 先入局的人能定义游戏规则。
4. **承诺装置：** 公开声明一个不给自己退路的约束（如MIT开源），让对手知道你没法变卦。

## 决策前自问

1. 对方会怎么动？——他不是静止的。
2. 规则决定了什么行为会被奖励？——看清激励机制。
3. 这是零和还是非零和？——不同游戏不同打法。
